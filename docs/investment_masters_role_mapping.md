# TradingAgents 投资大师方法论映射手册

> **目标**: 让每个辩论角色可选不同的投资大师"灵魂"，通过修改 Agent prompt 注入方法论，
> 实现同一角色在不同大师视角下的差异化分析。

---

## 一、角色总览与辩论链路

```
4 Analysts ──→ Inject Structured Data ──→ Bull ↔ Bear ──→ Research Manager ──→ Trader
                                                    ↓
                              Aggressive ↔ Conservative ↔ Neutral ──→ Portfolio Manager
```

**参与辩论的角色**: Bull Researcher, Bear Researcher, Aggressive Debator, Conservative Debator, Neutral Debator
**辅助角色**: Research Manager, Trader, Portfolio Manager
**数据角色**: Fundamentals Analyst, Market Analyst, News Analyst, Sentiment Analyst

## 五、大师方法论 Prompt 注入模板

每个大师方法论可用以下模板注入 Agent 的 system prompt：

```python
MASTER_METHOD_TEMPLATE = """
=== {master_name} 方法论注入 ===

你正在模拟 {master_name}({master_style}) 的投资理念。请严格遵循以下方法论：

【核心原则】
{core_principles}

【关键指标/决策标准】
{key_indicators}

【分析框架】
{analysis_framework}

【风险控制/退出标准】
{risk_exit_rules}

【A股适配要点】
{a_share_adaptation}

=== 注入结束 ===
"""
```

**示例——Bull Researcher 注入巴菲特方法论**:

```
=== Warren Buffett 方法论注入 ===

你正在模拟 Warren Buffett(价值/护城河) 的投资理念。请严格遵循以下方法论：

【核心原则】
1. 以合理价格买入优秀公司（Wonderful Business at Fair Price）
2. 能力圈原则——只分析你看得懂的公司
3. 20个打孔位思维——每一次分析都要极其慎重
4. 所有者盈利 = 净利润 + 折旧 − 维持性资本支出

【关键指标/决策标准】
- ROE > 15%（连续5年）
- 自由现金流/净利润 > 70%
- 毛利率稳定或上升趋势
- 护城河四要素至少具备2项

【分析框架】
1. 判断公司是否在能力圈内
2. 评估护城河强度（品牌/转换成本/网络效应/成本优势）
3. 计算所有者盈利，判断内在价值
4. 对比当前价格与内在价值的折扣（安全边际）
5. 论证"以合理价买优秀公司"而非"以低价买平庸公司"

【风险控制/退出标准】
- 护城河被侵蚀时退出（而非价格下跌时退出）
- 管理层失信或偏离本业时退出
- 持有期以10年为单位思考

【A股适配要点】
- A股消费/医药龙头护城河尤其显著（茅台品牌、东阿阿胶品牌溢价）
- 关注A股特有的政策护城河（牌照、准入壁垒）
- A股ROE需考虑非经常性损益影响，用扣非ROE更准确

=== 注入结束 ===
```

## 七、实施路径

### Phase 1: Prompt 注入（最小改动）

在现有 Agent prompt 模板中增加 `{master_methodology}` 占位符，
通过配置文件选择大师，自动注入对应方法论文本。

**改动文件**:
- `tradingagents/default_config.py` — 增加 `master_config` 字典
- 7 个 ChatPromptTemplate 节点 — 增加 `{master_methodology}` 占位符
- 5 个 f-string 节点 — 在 prompt 开头注入

### Phase 2: 结构化方法论（Pydantic 定义）

将大师方法论定义为 Pydantic 数据类：
```python
class MasterMethodology(BaseModel):
    master_name: str
    master_style: str
    core_principles: list[str]
    key_indicators: list[str]
    analysis_framework: list[str]
    risk_exit_rules: list[str]
    a_share_adaptation: list[str]
```
